# Space Invaders — DrRacket

A simplified version of the classic Space Invaders game, built using DrRacket 
and functional programming principles.

## Demo
[Watch Video Demo](https://youtu.be/QxkSsM7twjM)

## About
Invaders randomly fall from the sky while the player fires missiles from a tank 
in order to destroy them. The game ends when an invader reaches the ground.

## Controls
| Key | Action |
|-----|--------|
| ← Left Arrow | Move tank left |
| → Right Arrow | Move tank right |
| Space Bar | Fire missile |

## Tech Stack
- DrRacket
- Functional Programming paradigm

## How to Run
1. Install [DrRacket](https://racket-lang.org/)
2. Open `space-invaders-starter.rkt`
3. Click **Run**
